PayPal Adaptive payments handles payments between a sender of a payment and one or more receivers of the payment.

== Description ==
You are an application owner, such as a merchant that owns a website, the owner of a widget on a social networking site with buddypress, the provider of a payment application on mobile phones, and so on.

=Simple, Parallel, and Chained Payments=
Adaptive Payments provides several kinds of payment: simple, parallel, and chained payments.

== Installation ==

= Minimum Requirements =

* WordPress 4.3 or greater
* WooCommerce 2.4 or greater
* PHP version 5.6 or greater
* MySQL version 5.0 or greater

= Automatic installation =

Automatic installation is the easiest option as WordPress handles the file transfers itself and you donâ€™t need to leave your web browser. To do an automatic install of WooCommerce for PayPal Adaptive Payment in Japan, log in to your WordPress dashboard, navigate to the Plugins menu and click Add New.

= Manual installation =
The manual installation method involves downloading our plugin and uploading it to your webserver via your favourite FTP application.
